This repository is dedicated to those who intend to replicate my paper.
The folder BACI contains BACI datasets on trade data for each year from 2013 to 2023
The folder WITS contains yearly tariff schedules on four tariff types: MFN, AVE MFN, Preferential and AVE Preferential tariffs.
The file merge.ipynb contains script used to merge data
The file visuals.ipynb contains script used to make figures pasted in the paper
The file script.do contains Stata code to replicate product-level PPML regression for estimating elasticities.
